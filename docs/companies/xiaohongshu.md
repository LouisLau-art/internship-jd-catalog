## XHS-001 生态研发工程师实习生

- 类型：后端开发
- 地点：武汉市 / 北京市
- 推荐度：`strong`

### 工作职责

- 参与社区生态与安全业务方向的核心研发
- 与产品经理、算法工程师配合，参与需求讨论、功能定义
- 从设计到发布，亲手打造社区生态与安全业务相关系统能力
- 参与性能、架构、稳定性优化
- 探索 AI 在业务场景中的应用
- 建设审核系统自动化测试能力，提升服务稳定性

### 任职资格

- 计算机相关专业，27 年及之后毕业
- 熟悉至少一门面向对象语言，如 Java / Go / Python / C++
- 有后端技术经验加分，算法能力出色加分
- 每周可出勤 4-5 天，长期实习 6 个月以上
- 加分项：个人技术博客、GitHub 开源项目、关注业界技术分享

### 匹配原因

- `ScholarFlow` 和 `multi-cloud-email-sender` 可以支撑“需求到发布”的叙事
- 你已有博客和 GitHub，可直接吃到加分项
- “稳定性优化、自动化测试、跨角色协作”都能从现有经历里提炼
